# About the Modeling Healthcare Data With Graphs Course

Early in my career I was taught that the only way to store production data was using tables in a relational database.  Then in February of 2007 I learned the truth.  There were other ways of storing data that offered many advantages over tables.  This was the first step on a journey that changed my career.  I went on to start a conferences called the [NoSQL Now!]() conferences, I co-authored a book on NoSQL called [Making Sense of NoSQL](https://www.manning.com/books/making-sense-of-nosql), and I became a sought after consultant.
